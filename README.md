# نظام الاستخبارات الوبائية التفاعلي (Iraq Epi-Intelligence) 🇮🇶

مشروع نظام محاكاة وبائي متكامل يجمع بين قوة التحليل الإحصائي في لغة **R** ومرونة العرض التفاعلي في لغة **Python**.

## 🚀 ميزات المشروع
* **الربط بين اللغات (Interoperability):** استخدام R لتوليد البيانات (JSON) والتحليل الإحصائي، وPython لبناء الواجهة.
* **خريطة حرارية ديناميكية:** تمثيل مكاني للإصابات في 18 محافظة عراقية يتأثر بمستوى الحظر.
* **ميزان القرار:** أداة بصرية للمفاضلة بين الخسائر الاقتصادية والأرواح المنقذة.
* **التحليل الإحصائي:** رسوم بيانية متطورة باستخدام مكتبة `ggplot2`.

## 🛠️ التقنيات المستخدمة
* **Python:** (Streamlit, Folium, Pandas).
* **R:** (ggplot2, jsonlite) - للتحليل المسبق وتوحيد مصدر البيانات.
* **Data Format:** JSON (كجسر لنقل البيانات).

## 📂 هيكلية الملفات
* `epi_app.py`: ملف الواجهة الرئيسية.
* `epi_model.py`: كلاس الحسابات الرياضية والمنطق الوبائي.
* `iraq_analysis.R`: كود R لتوليد البيانات والرسوم الإحصائية.
* `cities_data.json`: قاعدة البيانات الناتجة عن R والمستخدمة في النظام.
